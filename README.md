# Comparaci-n-de-Programaci-n-
Este repositorio contiene dos versiones de un sistema básico de gestión de mascotas desarrollado en Python: Programación Tradicional y POO

Programación Tradicional

La solución se implementó utilizando variables y funciones. El programa solicita al usuario el nombre, la especie y la edad de una mascota, para posteriormente mostrar la información registrada de forma organizada.

Archivo principal:

programacion_tradicional/tradicional.py

Programación Orientada a Objetos (POO)

La solución se implementó utilizando una clase llamada Mascota, que contiene atributos y métodos para gestionar la información de las mascotas. Se crearon dos objetos de la clase y se ejecutaron los métodos definidos para mostrar la información y simular el sonido de cada mascota.

Archivos principales:

programacion_poo/mascota.py
programacion_poo/main.py
